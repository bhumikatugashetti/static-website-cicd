# Static Website CI/CD Pipeline

## Architecture
GitHub → AWS CodePipeline → S3 → Static Website Hosting

## Tools Used
- GitHub
- AWS CodePipeline
- Amazon S3

## Features
- Auto deployment on code push
- Static website hosting
- CI/CD automation

## How it works
1. Push code to GitHub main branch
2. AWS detects changes
3. Pipeline triggers automatically
4. Website updates in S3
